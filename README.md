# PTNK Chat
An Messenger Bot that allows auto-matching and random chat for HSG's students.
Try it out at: [PTNK Chat](https://m.me/ptnk.chat.2)

# Features
* ✅ Random chat
* ✅ Send stickers
* ✅ Send images, videos, files,...
* ✅ React to messages
